# fuquencer (Vue / Tone.js)

Working on a sequencer with Vue and Tone.js
Play @ https://zeye.ru/

## Project setup

```
yarn install
```

### Features

- [x] Basic 16 step sequencer
- [x] Basic bass synth
- [x] Basic drum samples
- [x] Auto generation of bass line
- [x] Per-track Effects
- [x] Additional per-track polyrythms
